# Design Handoff

This folder receives exports from Claude Design (claude.ai/design).

When Kevin exports a design bundle (zip), extract it here. The frontend-agent reads these files before building any screen.

## Convention
- Each export is a subfolder named by screen: `compose/`, `projects/`, `schedule/`, `reddit/`
- Files inside are the HTML/JSX prototypes from Claude Design
- The approved hi-fi for Compose is the V2 unified design (chip strip dissolves into card, hero gradient, green Approve pill)

## Current handoffs
- Compose - Hifi.html — approved V2 design (Spotify-energy dark theme)
- Wireframes - Compose.html — early wireframes (reference only)
